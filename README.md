### Env Variables

Create a .env file in root

<!-- Content -->

NODE_ENV = development
PORT = 5000
MONGO_URI = mongodb://localhost:27017
JWT_SECRET = 'abc123'

### Install Node Package (client & server)

npm install

cd client
npm install

### Load Database

npm run data:import

npm run data:destroy

### Run

npm run dev

# Just run server

npm run server
